Read the matrix from the Standard Input and a parameter d and output its PageRank with the pagerank method with a precision of 3 decimal places.

The first line contains two numbers n and d where n is the dimension of the matrix, an integer, and d is a damping parameter, a float. The next n lines contain n floats on each line representing a matrix row-by-row.
