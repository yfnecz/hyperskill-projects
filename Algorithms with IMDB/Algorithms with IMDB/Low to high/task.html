Description

Implementation of the linear search is straightforward and undemanding. It is also slow if you have a large dataset or when the item is at the end of a list. To improve time complexity, we need to implement a better algorithm. But first, let's talk about how to arrange data in either ascending or descending order based on the movie ratings.

In this stage, you need to sort the data by implementing bubble sort. This time, complexity is \(O(n^2)\), where \(n\) is the number of elements. Again, mind how much time it takes for your algorithm to execute.

Objectives

Your program should do the following:

  1. Compare the ratings of the first and second elements. If the rating of the first movie is greater than the second one, swap them;
  2. Next, compare the second and third elements. If the second one is greater than the third one, swap them. Continue comparing and swapping the elements until you reach the very end of the list;
  3. Repeat the above steps \(n-1\) times where \(n\) is the number of elements;
  4. Print the sorted data in the format as shown in the example.
Example

Example 1: a sorted array
