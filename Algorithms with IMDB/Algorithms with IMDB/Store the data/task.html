Description

Before we start creating algorithms, you need some input data to form a dataset. In our project, we are going to use a collection of movie titles and ratings from IMDB. It is a CSV file that can be downloaded from here: movies.csv.

It's also important to decide on the format to store the data. We will use it later to implement features for searching and sorting by ratings, so try to choose the format that will be easy to use. It's possible to pass this stage with any storage format, but you may want to change it in the later stages.

Objectives

To complete this stage, you need to perform the following steps:

  1. Download movies.csv;
  2. Read every line of the comma-separated data file. The first part is the movie title, and the second one is the corresponding movie rating;
  3. Store the data in a format of your liking. No quotes are expected in the final output, so make sure there are no quotes left in the movie titles;
  4. Print the data, separating the title and rating by a ' - ', as shown in the example. Print each movie on a new line.

Tip: Use encoding="UTF-8" for reading data from the file.
