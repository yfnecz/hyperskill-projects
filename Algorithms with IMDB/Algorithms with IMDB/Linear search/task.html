Description

The next step is to carry out a linear search over the resulting set from the previous stage. Implement a linear search algorithm that looks for movies with a rating of 6. Print the corresponding movie titles like in the Example.

The time complexity for the linear search is \(O(n)\), where \(n\) is the number of elements. Please, pay attention to how long it takes for your code with the linear search algorithm to execute over this dataset.

Objectives

In this stage, your program should perform the following steps:

  1. Iterate over data;
  2. Check if the rating of each element is equal to 6;
  3. If yes, print the movie title and the rating like in the previous stage.

Make sure you converted the ratings to float.

Tip: Don't forget that you can refer to functions for better code readability.

Example

Example 1: output of your program

Hercules - 6.0
The Fast and the Furious: Tokyo Drift - 6.0
Finding You - 6.0
